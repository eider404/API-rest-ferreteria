  • GET Obtener Determinado Producto

    Este endpoint devuelve una consulta de un producto específico mediante un campo y un valor segun el campo seleccionado

    • URL
    • https://api-ferreteria.fly.dev/consulta?campo=campo&valor=valor

    campo valor ejemplos Tipo de dato a usar
    id 4324 INT
    nombre Producto1 VARCHAR
    precio 342 INT

    • URL
    • https://api-ferreteria.fly.dev/consulta?campo=nombre&valor=Rotomartillo

    Codigo en JavaScript: 
    let campo="nombre";
let valor="Rotomartillo"

fetch( `https://api-ferreteria.fly.dev/consulta?campo=${campo}&valor=${valor}`)

  • POST Agregar Producto

    Este endpoint agrega los productos a la base de datos, dichos datos deben ser enviados desde el body

    • URL
    • https://api-ferreteria.fly.dev/new/

    Requisitos en los "headers"
    {
  'Content-Type': 'application/json'
}
    Requisitos para el "body"
    estructura del body tipo de dato a usar
    nombre VARCHAR
    precio INT
    img VARCHAR

    Codigo en JavaScript: 
    const obj = {
  "nombre": "Flexómetro",
  "precio": 200,
  "img": "https://www.gyptech.com.do/wp-content/uploads/CINTA-TRUPER-1800X1800-10M-e1592417473907.jpg",
}

fetch( `https://api-ferreteria.fly.dev/new/`, {
          method: 'POST',
          headers: {
              'Content-Type': 'application/json'
          },
          body: JSON.stringify( obj )
      })
      .then(res => res.json())
      .then( data =>{
          console.log(data)
      })
      .catch(err => console.log(err));

    Resultados: 
    {
  status: 200,
  mensaje: 'Producto Agregado',
  data: {
    nombre: 'Flexómetro',
    precio: 200,
    img: 'https://www.gyptech.com.do/wp-content/uploads/CINTA-TRUPER-1800X1800-10M-e1592417473907.jpg',
    fechaCreacion: '2023-02-25 04:18:36'
  },
  rows: {
    fieldCount: 0,
    affectedRows: 1,
    insertId: 32367,
    serverStatus: 2,
    warningCount: 0,
    message: '',
    protocol41: true,
    changedRows: 0
  }
}

  • PUT Actualizar Producto

    Este endpoint actualiza los productos a la base de datos, dichos datos deben ser enviados desde el body y el id en los parametros

    • URL
    • https://api-ferreteria.fly.dev/update/:id

    Requisitos en los "headers"
    {
  'Content-Type': 'application/json'
}
    Requisitos para el "body"
    estructura del body tipo de dato a usar
    id INT
    nombre VARCHAR
    precio INT
    img VARCHAR
    Requisitos para los "parametros"
    estructura de los parametros tipo de dato a usar
    id INT

    Codigo en JavaScript: 
    const obj = {
    id: 5,
    nombre: 'Flexómetro',
    precio: 200,
    img: 'https://www.gyptech.com.do/wp-content/uploads/CINTA-TRUPER-1800X1800-10M-e1592417473907.jpg',
}
  
fetch( `https://api-ferreteria.fly.dev/update/${obj.id}`, {
            method: 'PUT',
            headers: {
                'Content-Type': 'application/json'
            },
            body: JSON.stringify( obj )
        })
        .then(res => res.json())
        .then( data =>{
            console.log(data)
        })
        .catch(err => console.log(err));

    Resultados: 
    {
  status: 200,
  mensaje: 'Producto Actualizado',
  data: {
    id: '5',
    nombre: 'Flexómetro',
    precio: 200,
    img: 'https://www.gyptech.com.do/wp-content/uploads/CINTA-TRUPER-1800X1800-10M-e1592417473907.jpg',
    fechaCreacion: '2023-02-25 04:34:41'
  },
  rows: {
    fieldCount: 0,
    affectedRows: 1,
    insertId: 0,
    serverStatus: 2,
    warningCount: 0,
    message: '(Rows matched: 1  Changed: 1  Warnings: 0',
    protocol41: true,
    changedRows: 1
  }
}
